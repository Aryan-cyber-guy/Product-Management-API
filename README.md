<h1 align="center">📦 Product Management API</h1>

<p align="center">
A simple RESTful API built with <b>FastAPI</b> that allows users to manage products using CRUD operations and a SQL database.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project is a backend API for managing products in a database. 
It demonstrates how to build a REST API using FastAPI and SQLAlchemy.
</p>

<p>
The API allows users to:
</p>

<ul>
<li>View all products</li>
<li>Retrieve a specific product</li>
<li>Add new products</li>
<li>Update existing products</li>
<li>Delete products</li>
</ul>

<p>
The project also initializes the database with sample products when it starts.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
<li>📦 Product CRUD operations</li>
<li>🗄 Database integration using SQLAlchemy</li>
<li>⚡ FastAPI for high-performance APIs</li>
<li>📚 Automatic API documentation with Swagger</li>
<li>🌐 CORS support for frontend integration</li>
<li>🔄 Database initialization with sample products</li>
</ul>

<hr>

<h2>🛠 Tech Stack</h2>

<ul>
<li><b>Backend Framework:</b> <a href="https://fastapi.tiangolo.com/">FastAPI</a></li>
<li><b>Language:</b> Python</li>
<li><b>Database ORM:</b> <a href="https://www.sqlalchemy.org/">SQLAlchemy</a></li>
<li><b>Server:</b> Uvicorn</li>
<li><b>Data Validation:</b> Pydantic</li>
</ul>
